# 阶段四：动画与视觉效果

本阶段将深入学习 RmlUi 的动画系统，掌握如何创建流畅的过渡效果、关键帧动画和各种视觉特效，让你的界面更加生动和专业。

---

## 📖 本章内容

1. [CSS 过渡](01-transitions.md) - 过渡属性、缓动函数
2. [关键帧动画](02-keyframe-animation.md) - 动画定义和控制
3. [变换](03-transforms.md) - 平移、旋转、缩放
4. [滤镜效果](04-filters.md) - 模糊、亮度、对比度等
5. [渐变与阴影](05-gradients-shadows.md) - 背景渐变、盒阴影
6. [粒子特效](06-particle-effects.md) - 简单粒子系统

---

## 🎯 学习目标

完成本阶段后，你将能够：

- ✅ 使用 CSS 过渡创建平滑的状态变化
- ✅ 创建复杂的关键帧动画
- ✅ 运用 2D 变换制作各种视觉效果
- ✅ 应用滤镜效果增强视觉表现
- ✅ 使用渐变和阴影增加层次感
- ✅ 实现简单的粒子特效

---

## 📝 实践任务

### 任务 1：创建加载动画

设计一个加载指示器：
- 旋转的 loading 图标
- 脉冲效果
- 进度条动画

### 任务 2：制作按钮交互效果

实现丰富的按钮反馈：
- 悬停放大效果
- 点击波纹
- 禁用状态渐变

### 任务 3：创建页面转场

实现页面切换动画：
- 淡入淡出
- 滑动切换
- 缩放转场

### 任务 4：设计成就解锁动画

制作成就解锁特效：
- 图标弹入动画
- 光芒效果
- 粒子爆发
